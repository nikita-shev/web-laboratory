Чему вы научитесь?

Верстать при помощи HTML

  • Базовые теги HTML, их свойства и назначения.
  • Создание блочной верстки.
  • Изучение семантики код.
  • Научитесь размещать страницы в интернете (Хостинг и Домен).
  • Строить формы для отправки данных на сервере.

Создавать CSS стили

  • Сможете изменять свойства элементов страницы.
  • Строить стили используя их свойства и значения.
  • Научимся переносить стили из Photoshop в CSS.
  • Построите свой первый Landing Page по макету.

Базовые знания JavaScript

  • Базовые знания по программированию.
  • Понятие языка JavaScript и как он работает.
  • Получите понимание как работает программи- рование и как создаются программы.
  • Научитесь строить простые программы на языке JavaScript.
  • Сможете создавать анимацию и интерактив с пользователем, используя готовые библиотеки.

Профессиональные навыки

  • Photoshop - графический редактор для создания дизайна страниц.
  • Brackets - незаменимый редактор для верстальщика.
  • Emmet - плагин мгновенного написания кода.
  • Adobe Cloude - возможность верстки без Photoshop.
  • File Zilla - программа для работы с FTP.
  • Chorme Developer Tools - незаменимый инструмент для правки и изменения верстки в реальном времени.

Часто задаваемые вопросы

Это диск или DVD, как выглядит курс?

Вы покупаете доступ в личный кабинет, в котором у вас будут все записи, домашние задания и материалы курса. Вам будет присвоен личный логин и пароль, по которому вы сможете входить в любое время, с любой точки мира и обучаться. Главное чтобы был доступ в интернет.

На какой срок дается доступ в личный кабинет?

Видео-уроки доступны в течение всего курса. По окончании курса, если вы успешно сдали все домашние задания, у вас есть возможность получить видео-записи бесплатно

Для смены тарифного плана, вам необходимо доплатить сумму, недостающую до тарифа «Full». После этого с вами свяжется администратор курса и расскажет, что делать дальше.
